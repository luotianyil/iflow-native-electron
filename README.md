# iflow-native 

## 介绍

`iflow-native-electron` 是一套基于 `Electron` 以实现 `PHP` `Windows`、`Linux` 和 `Mac` 平台的桌面的端应用程序。

## 快速开始

```bash
git clone https://github.com/luotianyil/iflow-native-electron
```

## 依赖项目
```text
本项目用于实现PHP 窗口程序 需搭配以下 项目
```
